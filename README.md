# Telco Customer Churn Prediction

End-to-end ML project predicting customer churn for a telecom company.

## Results
- Model: Random Forest Classifier
- Accuracy: ~82%
- Key finding: Month-to-month customers churn 3x more than annual contract holders

## Business Recommendation
Target retention spend on month-to-month customers in their first 6 months
with upgrade incentives — highest ROI retention segment.

## Files
- `notebook.ipynb` — full EDA + model code
- `predictions.csv` — model predictions on test set
- `feature_importance.png` — top 10 churn drivers
- `eda_plots.png` — 4 business insight charts
